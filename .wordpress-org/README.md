## Freemius Dashboard Checklist
After creating the pages above:
1. Freemius dashboard → App Settings → Legal
2. Set Privacy Policy URL: https://fricking.website/privacy-policy
3. Set Terms of Service URL: https://fricking.website/terms
These URLs appear on the Freemius opt-in screen shown to users on first activation.
