AI-powered business insights scanner for Google Business Profiles using Claude AI.

== Description ==

F! Insights is an AI-powered business intelligence tool that helps you analyze Google Business Profiles, audit websites, and visualize competitive landscapes with interactive maps.

**Features include:**

* Google Business Profile analysis powered by Claude AI
* Competitor mapping with interactive Google Maps
* Lead capture and management
* Stored and shareable reports
* White-label customization (Pro)
* Market Intel analytics dashboard (Pro)

== Installation ==

1. Upload the `f-insights` folder to `/wp-content/plugins/`
2. Activate the plugin through the **Plugins** menu in WordPress
3. Go to **F Insights → Settings** and enter your Google Places API key and Claude API key
4. Add the shortcode `[f_insights]` to any page

== Frequently Asked Questions ==

= What API keys do I need? =

You need a Google Places API key and an Anthropic Claude API key.

= Does this work without a premium license? =

Yes. The free version includes the full scanner. Run 10 organic scans through your shortcode and a 30-day trial unlocks automatically — no card, no checkout. Premium adds analytics, white-labeling, email reports, lead management, and more.
